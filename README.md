# Eco Ideathon Solution
AI-Powered Rooftop Solar PV Detection (EcoInnovators 2026)

This repository contains an end-to-end AI pipeline for detecting rooftop solar panels, calculating their area, and verifying compliance with the PM Surya Ghar scheme using satellite imagery.

ğŸ§  Methodology

Model: YOLOv8-Large trained on ~10,000 satellite images.

Logic: Checks for solar panels within a 1200 sq.ft (primary) and 2400 sq.ft (secondary) buffer zone around the coordinate.

Quantification: Calculates area ($m^2$) based on pixel dimensions and GSD (0.15m/px).
